Keep it short and concrete.

## Project Outcome

`[2 to 4 sentences describing the positive impact of the solution]`

Suggested prompt:

organisations]`. It helps reduce `[pain, risk, delay, cost, confusion, or
manual work]` by `[how the solution changes the workflow]`. This creates value
through `[safety, clarity, privacy, efficiency, access, or better decisions]`.
