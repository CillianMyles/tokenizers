Organiser guidance captured from Sarah's submission email for the AI hackathon.

## Requirements

- Push the final code to the public GitHub repository by Tuesday, 14 April
  2026 at 12:30.
- Ensure the repository is public so judges can access it.
- Email the repository link to `sking@itag.ie`.
- Include the completed project description template in that email.

## Project Description Template

The submission template should include:

- a brief description of the project
- a project outcome describing the positive impact of the solution

This information will be shared with the judges before they review the project.
The organiser guidance emphasises concise, thoughtful writing so the judges can
quickly understand both the problem being solved and the impact of the
solution.
